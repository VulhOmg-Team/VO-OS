# VO-OS

## Vulhomg Operating System

Freedom for everyone. Opportunities for all. Together we build the future of Linux.

VO-OS is an open source Linux project created as a part of the global Linux community.

## Goal

To create a universal Linux platform:

- for beginners;
- for developers;
- for gamers;
- for creative professionals.

## Status

Project is currently in the planning and research stage.
